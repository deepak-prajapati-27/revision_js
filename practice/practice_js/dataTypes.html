DATA TYPES

What is Data Type ?

Datatype defines the kind of data we are storing in a variable.

There are two type of data types:

Premetive Datatypes:

In premetive we can only store single type of data in a variable.
There are seven types of Premetive Data types:

  1. Number
  2. String
  3. BigInt
  4. Boolean
  5. Null
  6. Undefined
  7. Symbol

Non-Premetive Datatypes:

In Non-Premetive we can store multiple type of data in a variable.
There are two types of non-premetive Data types:

  1. Array
  2. Object
