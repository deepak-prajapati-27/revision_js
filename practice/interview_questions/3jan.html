Q.3 What is Javascript Event ?

JavaScript event is an action or occurrence that takes place in the browser, such as a user clicking a button, resizing the window, or pressing a key.
These events trigger the execution of JavaScript code, allowing you to respond and perform specific actions.

Q.8 Define Data Types in js ?

Datatype defines the kind of data we are storing in a variable.

There are two type of data types:

Premetive Datatypes:

In premetive we can only store single type of data in a variable.
There are seven types of Premetive Data types:

  1. Number
  2. String
  3. BigInt
  4. Boolean
  5. Null
  6. Undefined
  7. Symbol

Non-Premetive Datatypes:

In Non-Premetive we can store multiple type of data in a variable.
There are two types of non-premetive Data types:

  1. Array
  2. Object

Q.11 What is Hosting in Js?

Hoisting refers to the process whereby the interpreter appears to move the declaration of functions, variables, classes, or imports to the top of their scope, prior to execution of the code.
